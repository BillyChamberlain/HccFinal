Editing

Make it yours without rebuilding the whole thing.

Change the timeline

Open assets/timeline-data.js. Add or remove objects in the timelineEvents array. The home page adapts automatically to the number of events.

Edit an event page

Each event links to a file in events/. Open any event HTML file and replace the title, copy, media blocks, or layout sections however you want.

Add another event

Duplicate one of the files in events/, rename it, then add a matching object to timeline-data.js with the new file path.
